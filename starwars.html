Introduction

Welcome to the swapi, the Star Wars API! This documentation should help you familiarise yourself with the resources available and how to consume them with HTTP requests.

API Details

Base URL: The root URL for all API requests is https://swapi.dev/api/

Rate Limiting: The API is limited to 10,000 requests per IP address per day to prevent abuse. This should be more than enough for any application.

Authentication: Swapi is a completely open API. No authentication is required to query and get data.

Encodings

JSON: This is the standard data format provided by default.

Wookiee: For our tall, hairy allies. This encoding is identical to JSON but with Wookiee translations. Use the toggle in the header to try it out! Appending ?format=wookiee to a URL enables it.

Helper Libraries

A variety of helper libraries are available for consuming the API in different languages:

  • Python: swapi-python
  • Javascript: SWAPI-Wrapper, swapi-node
  • Typescript: swapi-ts
  • Android: SWAPI-Android-SDK
  • Java: SWAPI
  • Go: swapi-go, swapi
  • PHP: Coruscant, swapi-php
  • Ruby: swapi-ruby, Tatooine, swgem
  • C Sharp: SharpTrooper, SWapiCSharp
  • And many more...
